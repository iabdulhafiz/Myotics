Overview of Myo Coding

The concepts of coding are all the same. No difference. It's just that language (or grammer, syntax) that changes. What's important is that you learn the basics of coding, so to simplify it for you, follow allong!

Rememberring things for later time (Variables)

We call them variables. Things that you store so that later you can use them. To store something, you need a name for it. Then put whatever you want to remember, it can be a number, a person's name, or even an action! The standard way to store something is to say, for example:

  • make "my age" equal to 13

After that, just call it by its name. Don't forget to use double quotations so the computer knows its your special item (variable).

Doing cool things (Functions)

Again, thanks to the people at P5js, the code can do some pretty cool stuff. The list bellow shows what actions currently work, what they need you to tell them, and and simple example:

  • Make background color:I can accept 3 colors, red green and blue from 0 to 255
    • make background color 34 red and 78 blue and 255 green
  • Draw a circle:I need the sideways position called x, the up-down posotion called y, and my width (radius)
    • draw a circle at x position of 23, y position of 90 and width of 43
  • Draw a rectangle (or square):I need the sideways position called x, the up-down posotion called y, and the width and height
    • draw a rectangle at position 34 in x and 73 in y and with width of 100 and height of 98
  • Change color of shape:I can accept 3 colors, red green and blue from 0 to 255
    • change shape color to 34 red and 78 blue and 255 green
  • Write text:I need the sideways position called x, the up-down posotion called y, and the text itself to write
    • write text "Welcome!" at x position of 23, y position of 90

Many more are soon to some! As we improve Myo Code, we will try to add more cool features so that you can make many more cool things!

Thinking before doing (Conditions)

These are if-statements, basically because we only do this code if that condition is met. Say you only want to draw a shape if the person's age is over 20. In this case, just give make an if statement before your action. Look at some examples so it makes sense:

  • if "my age" is greater than 13 then draw a circle at x position of 23, y position of 90 and width of 43

Remember that "my age" is the thing you saved before. It can be whatever you like as long as you saved it. Now when this line of code it run, it will see what was stored in "my age" and if it is greater than 13, the computer will draw the circle!

Why Does Myo Code Exists?

Apparent complexities in learning to code discourages many students from gaining knowledge and miss out the opportunity to challenge their reasoning and express their ideas. Having revolutionized the modern era, programming enhances critical knowledge, particularly role management and idea expression, that children face in current times. For this reason, we began development of an AI natural language processing system that enables beginners to begin coding in their own native language. The AI is constructed such that any language translated to English would then analyzed to execute the intended code. Consequently, this creates a zero-barrier entry point into the field of programming and hence enhances them in problem solving and design skills.

What Keeps Us Motivated?

We originally began as a team of undergraduate students at Ryerson University in Toronto, Canada. Having worked on design projects and capstone during our studies, we began realizing the need and importance of technology in and how it is rapidly changing the jobs of the future. For this reason, we founded Myotics which aims to be a world-provider for educational toys that enable kids to familiarize themselves with modern technology at a young age. With the help of you and those alike, we hope to make a difference in the living standards of generations to come.
